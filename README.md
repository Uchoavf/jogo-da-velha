# Jogo da Velha

Um clássico Jogo da Velha desenvolvido em Python com interface gráfica Tkinter.

### Funcionalidades
- Dois modos de jogo: **Vs IA** e **2 Jogadores**
- Dois níveis de IA: **Normal** (aleatório) e **Difícil** (algoritmo Minimax)
- Escolha entre jogar como **X** ou **O**
- **Placar** acumulado de vitórias durante a sessão
- **Destaque visual** da linha vencedora
- Botão **Desistir** e voltar ao **Menu** durante a partida
- Opção de **jogar novamente** no mesmo nível ao final

### Como jogar
```bash
python3 jogo_da_velha.py
```
